# Информационно-справочная система метрополитена.

## Описание проекта
Курсовая работа по дисциплине "Современные технологии программирования". 
Клиент-серверное приложение для построения маршрутов в метрополитене.

## Архитектура
- **Сервер:** Spring Boot приложение с REST API
- **Клиент:** JavaFX desktop приложение
- **База данных:** PostgreSQL/H2

## Стек технологий
- Java 17/21
- Spring Boot 3.x
- JavaFX
- PostgreSQL
- Maven
